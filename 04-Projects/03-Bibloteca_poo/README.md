# Biblioteca POO

A simple library management system built using **Object-Oriented Programming (OOP)** principles.

## 📚 Project Description

`biblioteca_poo` is a small project designed to simulate how a library works.
It allows managing books, users, and basic library operations using object-oriented concepts.

The goal of this project is to practice:

* Classes and objects
* Encapsulation
* Inheritance (if used)
* Basic program structure
* Clean and organized code

## ⚙️ Features

* Add new books to the library
* View available books
* Borrow books
* Return books
* Manage library records

## 🚀 How to Run

1. Clone the repository:

```
git clone https://github.com/yourusername/biblioteca_poo.git
```

2. Navigate into the project folder:

```
cd biblioteca_poo
```

3. Run the program:

```
python main.py
```
