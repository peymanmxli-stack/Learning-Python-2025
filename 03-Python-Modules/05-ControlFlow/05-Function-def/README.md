Functions — def (Function Definition)

Functions allow us to organize code, reuse logic, and build scalable programs.
The def keyword is the foundation of functional programming in Python.

A function is a named block of code designed to perform a specific task.

Instead of writing the same code repeatedly, we define it once and reuse it.

This module focuses only on:

How functions are defined

How parameters work

How execution flow enters a function

(❗ return is covered in the next module, separately.)
