A `while` loop in Python repeatedly executes a block of code **as long as a
condition remains True**.

Unlike `for` loops (which iterate over sequences), `while` loops are
**condition-driven**, making them ideal for:
- User input validation
- Repeating actions until a goal is reached
- Loops where the number of iterations is unknown

Understanding `while` loops is essential for writing **interactive,
dynamic, and responsive programs**.
