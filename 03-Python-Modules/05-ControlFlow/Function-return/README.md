Functions — return (Output & Control Flow)

The return statement is one of the most important concepts in Python programming.
It allows a function to send data back to the caller and control execution flow.

Without return, functions cannot produce reusable results — making real-world
software, calculations, validations, and data processing impossible.

This module focuses only on return, separated cleanly from def, to ensure
deep understanding and professional coding practices.
