## 📌 What Is a Module?

A **module** is simply a Python file (`.py`) that contains:

- Functions
- Variables
- Classes
- Executable statements

Any Python file can act as a module and be imported
into another Python file.

📄 Example:
```text
calculator.py  → module
main.py        → program using the module
🎯 Why Modules Are Important

In professional software development, modules are used to:

✔ Organize large programs
✔ Avoid code duplication
✔ Improve readability
✔ Enable teamwork
✔ Simplify debugging
✔ Make code reusable

Without modules, programs quickly become
large, messy, and difficult to maintain.

🧠 Types of Modules in Python

Python supports three main types of modules:

1️⃣ Built-in Modules

Modules included with Python by default.

Examples:

math

random

datetime

os

sys

No installation required ✅

2️⃣ User-Defined Modules

Modules created by the programmer.

These are normal .py files that you write
to organize your own code.

Used heavily in:

Applications

Libraries

University projects

Enterprise systems

3️⃣ Third-Party Modules

Modules created by the Python community and
installed using package managers like pip.

Examples:

numpy

pandas

requests

flask

Used for advanced and specialized tasks 🚀

🔗 Importing Modules (Overview)

Python provides several ways to import modules:

import module_name

from module_name import item

import module_name as alias

Each method is used depending on clarity,
convenience, and project standards.

(Detailed explanations are provided in notes and examples.)
